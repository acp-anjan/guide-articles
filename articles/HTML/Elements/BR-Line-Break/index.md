
## Br Line Break
The &lt;br&gt; element helps to break a line with in a paragraph. This element might be useful writing a peom or rhymes.
```
<p>
  Jonny! Jonny!<br>
  Yes Papa<br>
  Eating sugar?<br>
  No papa<br>
</p>
```
outputs:
  ```
  Jonny! Jonny!
  Yes Papa
  Eating sugar?
  No papa
 ```
